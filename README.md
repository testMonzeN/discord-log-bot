# Discord Bot

Это многофункциональный Discord-бот, разработанный для автоматизации и мониторинга событий на сервере. Бот предоставляет широкий набор функций, включая отслеживание событий, управление ролями, логирование и многое другое.

---

## Функциональность

Бот поддерживает следующие функции:

### 1. **Логирование событий**
   - Создание и удаление каналов.
   - Вход и выход пользователей из голосовых каналов.
   - Удаление и редактирование сообщений.
   - Выдача и снятие мута, бана, тайм-аута.
   - Получение и удаление ролей пользователями.
   - Создание и удаление приглашений.
   - Присоединение пользователей через приглашения.
   - Создание и удаление ролей.
   - Включение и отключение различных функций бота.

### 2. **Уведомления**
   - Уведомления о событиях в настроенный канал логов.
   - Красивые сообщения с использованием `Embed`.
   - Временные метки для каждого события.

### 3. **Управление каналом логов**
   - Администраторы могут настроить канал для логов с помощью команды `!setlog`.
   - Сообщения с командой `!setlog` автоматически удаляются.

### 4. **Отслеживание приглашений**
   - Бот отслеживает, через какое приглашение пользователь присоединился к серверу.
   - Уведомляет о создании и использовании приглашений.

### 5. **Отслеживание ролей**
   - Бот отслеживает создание и удаление ролей.
   - Указывает администратора, который создал или удалил роль.

## Установка

### 1. **Требования**
   - Python 3.8 или выше.
   - Библиотека `discord.py` (устанавливается через `pip`).
   - Токен вашего Discord-бота.

### 2. **Установка зависимостей**
   Установите необходимые зависимости с помощью команды:
   ```bash
   pip install discord.py
   ```

## **Управление функциями бота**

### 1. **setlog**
> - Обновлено:
> - !setlog-new - открытие интерфейса

> - В стиле команд:
> - !setlog-old — инструкция  
> - !setlog-old #канал all — настроить канал для всех функций.
> - !setlog-old #канал {*arg} — настроить канал для конкретной функции.
> - !setlog-old list — просмотреть текущие настройки каналов.
> - !setlog-old clear — очистить все настройки каналов.
> - !setlog-old #канал clear — очистить настройки для конкретного канала.


### 2. **!toggle {arg}**
- !toggle <function> — переключает состояние функции.
- !toggle list — показывает список всех функций и их состояние.
- !toggle on — включает все функции.  
- !toggle off — выключает все функции.
